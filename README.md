# Redesign Auto Estufa 4 Rodas 🏎️

Projeto de modernização do site  da oficina **Auto Estufa 4 Rodas**, focado em experiência do usuário, acessibilidade e conversão.

## 🚀 Sobre o Projeto
Este projeto surgiu da identificação de uma oportunidade real de melhoria no site antigo da empresa onde trabalho. O objetivo é criar uma interface moderna, totalmente responsiva e acessível.

### Principais Funcionalidades:
- **Página de Orçamento:** Formulário funcional para agilizar o atendimento.
- **Acessibilidade (PcD):** Estrutura pensada para leitores de tela e navegação facilitada.
- **Design Responsivo:** Foco total na experiência mobile (celular).

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Semântico)
- **CSS3** (Flexbox/Grid)
- **JavaScript** (Em desenvolvimento para lógica de formulários)

## 🚧 Status do Projeto
Atualmente estou finalizando a estrutura das páginas **Home** e **Orçamento**. O próximo passo é a implementação das validações e envio de dados com JavaScript.
